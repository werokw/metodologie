# movie_scripts_network_analysis

Szablon notatnika do zajec z humanistyki cyfrowej. Wersja studencka prowadzi od promptow w jezyku naturalnym do dzialajacego kodu w Google Colab, a wersja prowadzacego zawiera pelne rozwiazanie referencyjne.

## Pliki

- `analiza_sieciowa_scenariusza.ipynb` - wersja studencka bez gotowego kodu; kazdy etap to maly prompt + pusta komorka kodu + opis oczekiwanego wyniku.
- `analiza_sieciowa_scenariusza_prowadzacy.ipynb` - wersja prowadzacego z pelnym kodem referencyjnym (punkt odniesienia).

## Szybki start

1. Otworz `analiza_sieciowa_scenariusza.ipynb` w Google Colab.
2. Zmien adres scenariusza w komorce parametru.
3. Idz krok po kroku: skopiuj prompt do modelu AI, wklej kod, uruchom i porownaj wynik z opisem.
